**How it works**: Press **F5** anytime to **save**. It calls the game's own save routine — the same one autosave uses — which plays the notification sound, plus I added a brief *"Game saved"* message on screen. It respects the game's safety rules: *(won't save while loading, sleeping, in bed, or at the shipyard — you'll get a message saying why instead).*

**How to Install** - Put QuickSave.DLL into your Plugins folder in your BepInEx folder.
**example** C:\Users\(name)\AppData\Roaming\Thunderstore Mod Manager\DataFolder\Sailwind\profiles\(name)\BepInEx\plugins
